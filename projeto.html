Sobre nós
Projeto

Justificativa

Nota-se que diversos estudantes apresentam diversas dificuldades no componente curricular de física e suas fórmulas e cálculos complexos, gerando um atraso no aprendizado geral da turma e uma sobrecarga no professor por conta da quantidade de dúvidas. Portanto, o presente projeto visa criar a calculadora para auxiliar e facilitar o processo de ensino e aprendizagem dos alunos. As explicações disponibilizadas sobre as fórmulas irão auxiliar na compreensão fundamentada do conteúdo.

Metodologia

Neste projeto, será utilizado o Front-End para customizar o site, dando cores e uma marca própria para ele, o Back-End, para os cálculos, juntamente com as fórmulas de física e o Banco de Dados para armazenar o conceito e legenda de cada fórmula, assim facilitando o entendimento do usuário.

Tecnologias Usadas

HTML, CSS, JS, PHP, MySQL Query Browser, GitHub, Figma, BR Modelo Web

Objetivo

Criar um software que, além de resolver os cálculos básicos, auxilie em fórmulas da física referentes aos conteúdos do 1° e 2° ano, mostrando o conceito e uma legenda explicando a fórmula o significado de cada incógnita. 

Introdução

Utilizando linguagens de programação e banco de dados, será desenvolvido um software de cálculos e fórmulas físicas com o objetivo de facilitar e auxiliar os alunos e os professores durante as aulas.
